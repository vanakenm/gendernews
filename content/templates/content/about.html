Questions fréquemment posées

Le site analyse un groupe de sites de nouvelles belges et tente d'identifier tout les noms de personnes citées avec leur genre. Les résultats que vous voyez donne une idée de la diversité de genre des nouvelles pour chaque site.

Nous pensons que la représentation est un élément important de la diversité - si toutes les nouvelles ne parlent jamais que d'hommes, elles vont contribuer à l'image d'un monde "par et pour les hommes".

Nous sommes intéressé par faire le même exercices sur les sites néérlandophones. Il va nous falloir de l'aide pour la traduction - si vous êtes intéressé, faite le nous savoir via le formulaire de contact. Merci !

Bien sur que non ! Mais comme dans toute initiative, il faut commençer quelque part.

Il y a un petit travail à faire de notre côté pour ajouter une source de données supplémentaire, mais c'est assez rapide. Vous en voyez une qui devrait être rajoutée ? Faites le nous savoir via la page de contact.

Le site est codé en Django. Les analyses sont faites grâce l'algorithme NER de Standford, via le package Spacy. Le code complet est disponible sur GitHub (un répertoire de code), plus spécifiquement ici: https://github.com/vanakenm/gendernews

Merci de l'envoyer via la page de contact.
